캐시플레이스 베타 테스터 모집
실내위치 플랫폼을 개발한 로플랫이 매장 정보를 수집하기 위해 모바일 앱 캐시플레이스를 선보였습니다.
캐시플레이스는 매장 방문 시 사진을 찍어 업로드하면, 100원~500원 수준의 리워드가 적립되는 앱입니다.
불쾌한 광고나 지루한 과정 없이, 카페, 상점, 식당 등을 오가면서 생각날 때 매장 내부 사진을 찰칵 찍기만 하면 됩니다.
캐시플레이스는 오는 9월 정식 서비스를 시작할 예정이며, 현재 베타테스터를 모집하고 있습니다.
많은 참여와 관심 보내주시기 바랍니다.
